ตัวแปรไม่สามารถใช้งานได้จนกว่าจะประกาศตัวแปร 
เช่น ถ้าต้องการใช้ตัวแปรจำนวน ชื่อ i สามารถประกาศดังนี้
int c;
ชื่อตัวแปร
    ประกอบด้วยอักขระและตัวเลขโดยตัวแรกของชื่อต้องเป็นอักขระ สามารถใช้ _ ได้ซึ่งช่วย
ให้สามารถอ่านโค๊ดได้ง่ายขึ้น ตัวอักขระภาษาอังกฤษพิมพ์ใหญ่และพิมพ์เล็กถือว่าแตกต่างกัน

การตั้งชื่อตัวแปร
    ควรตั้งให้อ่านแล้วเข้าใจได้ง่ายโดยใช้หลักการเดียวกันตั้งชื่อฟังก์ชันได้ด้วย

ประเภทข้อมูลและขนาด
- ประเภทข้อมูลที่ใช้เก็บเลขจำนวนเต็ม
    char ขนาด 1 ไบท์
    int ขนาด 4 ไบท์
    float ขนาด 4 ไบท์
    double ขนาด 8 ไบท์
- คีย์เวิร์ด short ลดขนาดเก็บ long ขยายขนาดเก็บ
    short int
    long int
- คีย์เวิร์ด signed ใช้กำหนดตัวแปรจำนวนเต็มให้เก็บเครื่องหมาย ส่วน unsigned เก็บเฉพาะค่าบวก
    signed int (ไม่ใส่ก็ให้ผลเหมือนกัน)
    unsigned char
- ใช้ sizeof ซึ่งเป็นการเช็คขนาดของตัวแปรและประเภทข้อมูล

printf()
- format ของ printf()
    %d or %i เลขจำนวนเต็มฐานสิบ
    %6d เลขจำนวนเต็มฐานสิบที่มีความกว้างอย่างน้อย 6 ตัวอักษร(ถ้าไม่มีจะเว้นช่องว่างไว้)
    %06d เลขจำนวนเต็มฐานสิบที่มีความกว้างอย่างน้อย 6 ตัวอักษร(ถ้าไม่มีจะใส่เลข 0 แทน)
    %f เลขทศนิยม
    %6f เลขทศนิยมที่มีความกว้างอย่างน้อย 6 ตัวอักษร
    %.2f เลขทศนิยมที่มีความกว้างอย่างน้อย 2 ตัวอักษรหลังจุดทศนิยม
    %6.2f เลขทศนิยมที่มีความกว้างอย่างน้อย 6 ตัวอักษร และความกว้าง 2 ตัวอักษรหลังจุดทศนิยม

- รูปแบบอื่นที่ใช้ในบางกรณี
    %o เลขฐาแปด
    %x เลขฐานสิบหก
    %c ตคัวอักษร
    %s สตริงของตัวอักษร
    %% สำหรับ % เอง

Character Input and Output
รูปแบบ
#include <stdio.h>
int getchar();
int putchar(int c);
- c = getchar() ใช้สำหรับรับอักขระ 1 ตัวจาก stdin แล้วนำมากำหนดค่าในโปรแกรม
- putchar(c) ใช้สำหรับส่งที่มีค่าใน c ออกไปยัง stdout

นิพจน์
    ในภาษาโปรแกรมโดยทั่วไป นิพจน์จะอยู่ในรูปของการรวมกันของสัญลักษณ์และเครื่องหมาย
เพื่อให้สามารถอ้างอิงถึงค่าข้อมูลได้ โดยปกติแล้วนิพจน์จะประกอบด้วย 1 operator 1 operand
แต่นิพจน์สามารถอยู่ในรูปค่าคงที่ก็ได้

<< left shift
>> right shift
~ one's complement(unary)
| or
