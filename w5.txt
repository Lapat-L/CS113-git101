Scope Rules (ขอบเขตและช่วงชีวิตของตัวแปร)
1.Local Variables ถูกประกาศไว้ใน block ซึ่งสามารถใช้ได้แค่ในฟังก์ชันที่ประกาศ
2.Global Variables ถูกประกาศไว้นอกฟังก์ชัน ถ้า Local ชื่อซ้ำกับ Global ค่านั้นจะไม่ส่งผลต่อ
ค่า Global
3.Dynamic Storage Duration เป็นพื้นที่เก็บข้อมูลสามารถถูก allocate ขึ้นได้ตามความต้องการ
ของโปรแกรมขณะที่โปรแกรมกำลังทำงาน

การคอมไพล์แยกส่วน
เช่น มี โปรแกรม main.c กับ func.c ถ้าต้องการให้นำสองโปรแกรมมาเชื่อมกันให้คอมไพล์ดังนี้
$ gcc main.c func.c -o main.exe
จะได้โปรแกรม main.exe ที่มีโปรแกรมของ main.c func.c เชื่อมกัน

Random Number Generator
- rand() เป็นฟังก์ชันใน C Standard Library ที่ใช้ในการสร้างเลขสุ่มที่อยู่ระหว่าง 0
ถึง Rand_max ซึ่ง Rand_max เป็นค่าคงที่ที่ถูกกำหนดใน <stdlib.h>
- ถ้าในโปรแกรมมีการเรียกใช้ rand() หลายครั้ง ค่าที่ได้ในแต่ละครั้งจะแตกต่างกันออกไป
- สามารถใช้ srand() เพื่อกำหนดค่า seed ของ rand() ได้
